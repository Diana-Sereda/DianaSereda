About me

Whenever I saw a clumsy website on the Internet, I always asked myself: "Is it so difficult to do it in a high quality?" That's how I started my journey in website development.

Desire for new knowledge and perseverance are the qualities that can characterize me. Everything I have already learned, I laboriously searched through all the available sources and did not give up when it did not work out. Studying on my own, I obtained a great experience, but I feel that I got stuck... To move on, I need a mentor and a team next to whom I will improve my skills.

I speak three languages, love the atmosphere of libraries and nights with canvas and brushes.

Mapty

Source code

Noncommercial project

Mapty app was done as project work in the 'The complete JavaScript course 2022' on the Udemy platform.
This app allows users to load the workouts and check the progress. It has a sidebar on the left side and a map. The map is loaded from the third-party service. Position on it,is automatically obtained by the browser using geolocation. By clicking on the specific location on the map, the form on the sidebar shows. The form allows to save this workout's details. After the submitting form user can see the tag with main info directly on the map.

Forkify

Source code

Noncommercial project

Forkify is a recipe application that loads recipes using API and renders them.This app contains:
- a search bar,
- a list of all recipes found on your query, divided by pages (each page contains 10 recipes),
- the main part with rendered recipes that you are interested in,
- a bookmarks section that contains all the bookmarked recipes
- "Add custom recipe" pop-up window.
If the recipe is added by you, it automatically marked by “client icon,”. If you bookmark recipe it is marked by “bookmark icon”.
This application uses classes, modules, promises, async/await, parcel and more.
