BMPT

Source code

Commercial project

BMPT is my first large-scale project requested by a customer from Austria. It is a platform for finding a nutritionist or trainer for any type of sport all around the world. The project contains such blocks as information about trainers/ nutritionists, personal accounts of both the trainer and the user, a blog page, etc.
In the section personal account, users can write posts that will be published in the application itself.
The BMPT is written in Vue 3 js using Vue-router, vuex and vue components, vue-star-rating, vue3-carousel, vue-countries-cities and sass. Moreover, it is also fully adapted to various devices.

Mapty

Source code

Noncommercial project

Mapty app was done as a project work in the 'The complete JavaScript course 2022' on the Udemy platform. This app allows users to load the workouts and check the progress. It has a sidebar on the left side and a map. The map is loaded from the third-party service. Position on it, is automatically obtained by the browser using geolocation. By clicking on the specific location on the map, the form on the sidebar shows. The form allows to save this workout's details. After the submitting form, user can see the tag with main info directly on the map.

Forkify

Source code

Noncommercial project

Forkify is a recipe application that loads recipes using API and renders them.This app contains:
- a search bar,
- a list of all recipes found on your query, divided by pages (each page contains 10 recipes),
- the main part with rendered recipes that you are interested in,
- a bookmarks section that contains all the bookmarked recipes,
- a "Add custom recipe" pop-up window.

If the recipe is added by you,it automatically marked by a “client icon,”. If you bookmark recipe, is marked by a “bookmark icon”.
This application uses classes, modules, promises, async/await, parcel and more.
